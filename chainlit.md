# Quant Pitch Evaluator

Use this chat to submit a quantitative stock pitch.

Required for evaluation:
- thesis
- time horizon (`days`, `weeks`, `months`, `years`)
- stock ticker(s) (submit just the tickers, e.g. `AAPL, MSFT`)
- methodology summary
- source URL(s) for submitted data

Commands:
- `/status` check readiness
- `/checklist` show onboarding checklist
- `/evaluate` run v0 score + allocation
- `/reset` start over

To upload files, use the attachment button in the chat input.
